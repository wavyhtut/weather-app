# Weather App

A simple web application that allows users to search for a city and view real-time weather information using an external API.

## Features
- Search weather by city
- Retrieve real-time weather data from an API
- Display weather information dynamically on the page

## Technologies Used
- HTML
- CSS
- JavaScript
- REST API

## Setup
This project requires an API key.

1. Obtain an API key from the [weather API provider](https://openweathermap.org/)
2. Replace `YOUR_API_KEY` in the source code with your own key
